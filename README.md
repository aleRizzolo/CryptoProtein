## What is CryptoProtein

CryptoProtein is a Web3 protein store <br>
It uses BlockChain and ETH to make a transaction

## How to run it

### Prerequisite

1. Ganache
2. Metamask
3. Node.js and npm

<code>truffle compile</code>
<code>truffle migrate</code>
<code>npm run start</code>
